## TODO.GG는 태스크관리와 게임 전적 조회 기능을 접목시킨 협업툴입니다.

달성도에 따라 프로젝트 상태를 업데이트함으로써 

성취감을 고양시키고 업무를 게임처럼 재미있게 수행할 수 있습니다 !
